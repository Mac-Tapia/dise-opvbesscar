================================================================================

FASE 1: Instalar Python 3.11 (5 minutos)
  1. Descarga desde https://www.python.org/downloads/
  2. Instala con "Add to PATH" MARCADO
  3. Reabre PowerShell
  4. Verifica: python3.11 --version

FASE 2: Crear venv (2 minutos)
  1. python3.11 -m venv .venv
  2. .\.venv\Scripts\Activate.ps1
  3. Verifica: python --version muestre 3.11.x

FASE 3: Instalar dependencias (5 minutos)
  1. pip install -r requirements-training.txt
  2. Espera a completar

FASE 4: Verificar todo (1 minuto)
  1. python verificacion_final_listo.py
  2. Debe mostrar: TODOS LOS SISTEMAS LISTOS

FASE 5: Ejecutar A2C (30-60 minutos con GPU)
  1. python -m scripts.run_a2c_only --config configs/default.yaml
  2. Entrena agente A2C
